已经能正常启动的板可以跳过本节，直接到 2.3 接入设备

通过引导向导烧录 ​ 首次登录后，新手引导向导在"选择硬件"完成后会自动进入 烧录系统 步骤，列出与你选中板型匹配的推荐镜像、一键打开烧录工具： 点击 打开烧录工具 后会跳到完整的烧录向导（ 系统烧录 tab），整个流程分四步： 选择设备 → 选择镜像 → 开始烧录 → 完成

按以下顺序操作： 选择目标板型（RDK X3 / RDK X5 / RDK S100） 选择推荐镜像版本（默认列出 D-Robotics 官方镜像） 确认目标存储介质（TF 卡、eMMC 或专用工具） 等待下载与烧录完成 烧录耗时与介质和镜像大小有关，TF 卡烧录通常 5~15 分钟

如果错过引导向导，可以在 系统烧录 tab 重新进入烧录流程

三种烧录介质的快速对照 ​ 板型 推荐烧录方式 RDK X3 TF 卡烧录 RDK X5（不带 eMMC） TF 卡烧录 RDK X5（带 eMMC 版本） TF 卡烧录 或 eMMC 烧录 RDK S100 专用 xburn 工具 烧录中的注意事项 ​ 烧录过程中可以切换到其他 tab 查看 AI 对话或运行命令，但不能： 拔出 TF 卡或断开 USB 连接 让 PC 进入睡眠或休眠状态 关闭 Studio 主窗口（最小化是可以的） 烧录完成后将 TF 卡插入板端、给板上电，等待板启动

详细操作与高级用法 ​ 本节只提供"快速跑通"所需的最简流程

下列高级场景请查阅 3.7 系统烧录 ： TF 卡烧录的 balanced 与 turbo 模式 上传自定义本地镜像 eMMC 烧录与 eMMC 备份 RDK S100 xburn 详细操作 烧录失败的完整排查清单 后续操作 ​ 板烧录完成并通电启动后，进入 2.3 接入设备 把板加入 Studio 的设备列表

Frequently Asked Questions Troubleshooting for common issues: SSH, Type-C flash, camera, model loading, WiFi, remote IDE, multi-device switching, token usage, model quality, and VNC or desktop latency. 📄️ 5.1 AI Assistant Not Responding Typical Symptoms 📄️ 5.2 SSH Connection Failure Typical Symptoms 📄️ 5.3 Type-C Flash Connect Failure Typical Symptoms 📄️ 5.4 No Image from USB Camera Typical Symptoms 📄️ 5.5 HBM Model Fails to Load Typical Symptoms 📄️ 5.6 OpenClaw Installation Failure Typical Symptoms 📄️ 5.7 WiFi Connection Failure Typical Symptoms: After entering the SSID and password in the WiFi Configuration pop-up window, connection fails / connection appears successful but ip addr shows that wlan0 hasn&#x27;t obtained an IP address / WiFi configuration is lost after reboot. 📄️ 5.8 Remote IDE Fails to Start Typical symptoms: After clicking "Open Remote IDE" in the IDE tab, the browser shows a blank page, displays an error code-server not installed, or gets stuck at "Downloading deb package" during installation. 📄️ 5.9 Multi-device Switching Issues Typical symptoms: After switching devices, the AI conversation context appears to remain on the previous device / the "device list" is missing / two boards are connected simultaneously, and running device_exec targets the wrong device. 📄️ 5.10 Abnormal Token Usage Typical symptoms: After sending just a few messages, the token capsule at the top of AI Dock turns red; monthly bills far exceed budget. 📄️ 5.11 Model Response Quality Issues Typical symptoms: Off-topic responses / repetitive outputs / incorrect tool selection / raw ... tags appearing in output / Chinese responses mixed with English (or vice versa). 📄️ 5.12 Remote Desktop Connection Issues or Lag Typical symptoms: The Remote Desktop tab keeps spinning without connecting / noticeable delay (>2 seconds) after connection / mouse clicks register at incorrect positions / only half the screen is displayed / black screen.
